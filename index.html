.zshrc Generator

zsh is a very configurable shell, but the myriad of configuration options is overwhelming to most. Below is a .zshrc generator that should hit the most important knobs without going too into the weeds. When in doubt the first option is generally acceptable.

The last box is the generated .zshrc which should be copied into .zshrc in your home directory.

Naturally this generator omits most options and context specific configurability. For more details see man zshoptions for more on options and man zshcompsys for more on completion system styles.
